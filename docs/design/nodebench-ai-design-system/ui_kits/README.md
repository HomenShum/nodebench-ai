# UI kits — canonical location

The interactive UI kits below are mirrored at the repo root for direct hosting and demo use:

```
/ui_kits/nodebench-web/      ← flagship web app
/ui_kits/nodebench-mobile/   ← iOS-shell mobile prototype
/ui_kits/nodebench-mcp/      ← terminal/MCP visual lane
```

The `nodebench-workspace/` kit only lives here — it predates the design packet
and is not part of the latest handoff.

When refreshing from a new design packet from `claude.ai/design`, refresh
**both** locations together. See `docs/runbooks/PROD_PARITY_UI_KIT_WORKFLOW.md`
for the full workflow.

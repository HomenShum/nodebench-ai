## PRODUCTION NOTES

### Why this works (TikTok/Manus pattern)
- **Zero preamble**: No "meet Sarah" or "imagine a world where." The product IS the hook.
- **Input-output clarity**: Viewer sees exactly what you type and exactly what you get.
- **Speed as wow**: The streaming Decision Memo appearing in real-time is the dopamine hit.
- **Rewatchable**: At 42 seconds, viewers will loop it. Each loop they notice more detail.
- **Screenshot-friendly**: The full Decision Memo frame (0:32) is the viral screenshot.

### Vertical cut (9:16) notes
- Crop to agent panel only — the sidebar and dashboard are not visible.
- The pull-back at 0:20 is replaced with a scroll-down to reveal the full memo.
- CTA text is larger (readable on phone).

### Frames needed
| Frame | Source |
|-------|--------|
| Black + question text | Generated (title card) |
| Agent panel input | Screenshot of `/?surface=ask` with panel open |
| Agent panel streaming | Animated capture or multiple frames simulating stream |
| Full dashboard reveal | Screenshot of `/founder` with agent panel overlay |
| Confidence score zoom | Cropped from Decision Memo section |
| CTA card | Generated (title card) |

### Key difference from original
The original is a 4:25 narrated slideshow. This variant has **12 seconds of speech** and lets the product's visual output do all the persuasion. The bet: a Decision Memo appearing in real-time is more compelling than any narration about it.

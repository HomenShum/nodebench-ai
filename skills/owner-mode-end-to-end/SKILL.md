# Skill: Owner Mode End-to-End

Use this skill when the task spans multiple layers and the agent needs to operate like the accountable owner instead of a narrow implementer.

Follow:

1. contract and data model
2. backend behavior
3. frontend or operator surface
4. exact verdict or status surfacing
5. deterministic verification
6. docs, rules, and skill updates when the workflow changes
